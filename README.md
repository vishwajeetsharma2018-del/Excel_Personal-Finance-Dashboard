# Excel_Personal-Finance-Dashboard
Personal Finance Dashboard built using Microsoft Excel to track income, expenses, and savings. The dashboard provides category-wise analysis, monthly summaries, and interactive charts to help users understand spending patterns and manage finances effectively. Demonstrates data cleaning, visualization, and Excel dashboard skills.
